## About

POPO는 PoApper에서 개발하고, POSTECH 총학생회에서 운영하는 포털 사이트입니다. POPO를 통해 교내의 장소/장비를 예약하고, 자치단체 및 동아리 정보를 열람할 수 있습니다.


## Setup

``` bash
$ npm install
// fill correct envrionments variables to .env file
$ npm start
```

## Stack

### Interpreter

- `node >= 16.13.2`
- `npm >= 8.3.1`

### Framework

- `NextJS >= 12.0.7`

### CSS

- `Semantic-ui-react`

### Runner

- AWS EC2
